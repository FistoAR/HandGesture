✦ HAND GESTURES

✋✋ Two-Hand Controls (Priority)

🤲
HANDS APART → EXPLODE
Move both hands away from each other to explode the model.
🙏
HANDS TOGETHER → IMPLODE
Bring both hands closer together to assemble the model.
⟷
BOTH HANDS MOVE → ROTATE
Move both hands in same direction to rotate model.
↕
BOTH HANDS UP/DOWN → ZOOM
Both hands up = zoom in, both down = zoom out.

☝️ Single-Hand Controls (Fallback)

🕷️
SPIDER HAND → ROTATE
All 5 fingers curved like a claw, spread apart. Move hand to rotate.
✌️
TWO FINGERS → ZOOM
Index + middle finger up. Spread apart = zoom in, close = zoom out.
🤏
PINCH → RESET
Thumb touches index fingertip. Resets everything.
